Следуя простым шагам, вы сможете быстро освоится в данном веб-приложении

Инструкция содержит описание полей и требований, предъявляемых к корректности вводимых данных, описание возможностей, схем и механизмов переходов по веб-страницам.

Главная страница

На странице реализованы возможности:

  • Переход к главной странице модели "Склады";
  • Переход к главной странице модели "Товары";
  • Вывод сводного отчета, содержащего связанные данные из двух моделей в формате: склад и ниже - список товаров и их некоторых характеристик (для каждого склада).

Склады

На главной странице реализованы возможности:

  • Создание записи о новом складе;
  • Переход к отчёту, который содержит полные данные о складах;
  • Просмотр списка складов с возможностью перехода к дополнительной информации о выбранном элементе.

Правила ввода данных:

Поле Пояснение (назначение) Тип данных Правила ввода
Название склада Предназначено для указания названия склада Текст Значение должно быть уникальным
Номер платформы Предназначено для указания номера платформы разгрузки товаров на склад Целое число Значение должно быть неотрицательным числом
ФИО ответсвенного Предназначено для указания ФИО ответственного за склад Текст Необязательное поле
Адрес Предназначено для указания адреса склада Текст Необязательное поле
Площадь Предназначено для указания площади склада Целое число Значение должно быть неотрицательным числом

Товары

На главной странице реализованы возможности:

  • Создание записи о новом товаре;
  • Переход к отчёту, который содержит полные данные о товарах;
  • Просмотр списка товаров с возможностью перехода к дополнительной информации о выбранном элементе.

Правила ввода данных:

Поле Пояснение (назначение) Тип данных Правила ввода
Наименование товара Предназначено для указания наименования товара Текст Значение должно быть уникальным
Категория Предназначено для указания категории товара (из списка категорий: Мебель, Стройматериалы (Строительные материалы), Инструменты (Инструменты для выполнения строительных,отделочных работ), Техника (Технические средства, машины для строительных, отделочных работ)) Текст Значение должно быть выбрано из предоставленного списка
Количество Предназначено для указания количества товара Целое число Значение должно быть неотрицательным числом
Дней хранения Предназначено для указания срока, в течении которого товар будет хранится на складе, в днях Целое число Значение должно быть неотрицательным числом
Описание Предназначено для указания описания к товару Текст Необязательное поле
Склад Предназначено для указания склада (из списка существующих складов), на котором хранится товар (на одном складе может быть много товаров) Текст Значение должно быть выбрано из предоставленного списка
Модуль склада Предназначено для указания модуля склада (иерархия места товара на складе: модуль-секция-ряд-место)(возможные значения: 'А', 'Б', ..., 'З') Текст Необязательное поле. Значение должно быть выбрано из предоставленного списка
Секция модуля склада Предназначено для указания секции модуля склада (возможные значения: 'S1', 'S2', ..., 'S16') Текст Необязательное поле. Значение должно быть выбрано из предоставленного списка
Ряд секции Предназначено для указания ряда секции модуля склада (возможные значения: '1', '2', ..., '32') Текст Необязательное поле. Значение должно быть выбрано из предоставленного списка
Место в ряду Предназначено для указания место в ряду секции модуля склада (возможные значения: 'M1', 'M2', ..., 'M128') Текст Необязательное поле. Значение должно быть выбрано из предоставленного списка

Детальный просмотр элемента

Просмотр "страницы" элемента модели, которая содержит:

  • Все параметры элемента (если параметр - необязательный для указания и не был заполнен при создании или редактировании - напротив него будет выведено "Данные не указаны");
  • Кнопка "Удалить" которая переведет на страницу удаления текущего элемента (подробнее в этом пункте);
  • Кнопка "Редактировать" которая переведет на страницу изменения параметров текущего элемента (подробнее в этом пункте);

Создание, редактирование и удаление записей

Для каждой модели данных (склады и товары) реализованы функции создания, редактирования и удаления записей:

Создание доступно по кнопке на соответствующей главной странице модели данных.

  • После ввода данных в все необходимые для заполнения поля соответвующей модели, нажать кнопку "Добавить". После этого будет добавлена новая запись.

Редактирование и удаление доступно по кнопкам в детальном описании элементов моделей данных (на главной страницы модели данных, в списке всех элементов по нажатию на кнопку "Подробнее" в соответсвующей строке элемента)

Редактирование:

  • После редактирования данных нажать кнопку "Обновить". После этого выбранная запись будет обновлена.

Удаление:

  • По умолчанию ползователь согласен на удаление выбранных данных (галочкой отмечен пункт "Я подтверждаю, что хочу удалить запись.", если галочка не стоит, пользователь удалить запись не может). После нажатия на кнопку "Удалить" и выбранная запись будет удалена.
